# FLUX Todo Example App

## Usage / Installation

To install everything, compile the test files, and test the app, use the following:

```
npm install -g webpack
npm install
webpack-dev-server
```

Open a new terminal and run
```
node services.js
```

Then go to `localhost:8080` in your browser.
